# MVP Technical Document

## Overview

The MVP (Minimum Viable Product) for the institute website focuses on delivering essential features needed for students and parents to access information and submit enquiries.

The goal is to launch a simple but functional website quickly.

---

## MVP Features

The MVP will include the following features:

1. Homepage
2. Course / Career Programs
3. Faculty Information
4. Results Section
5. Registration Form
6. Enquiry Form
7. Reviews / Testimonials

---

## Technology Stack

Frontend:

- HTML
- CSS
- JavaScript
- Bootstrap

Optional Frontend:

- React

Form Handling:

- Google Forms

Data Storage:

- Google Sheets

Deployment:

- Netlify / GitHub Pages

---

## Key Components

### Homepage
Displays institute overview, highlights, and quick navigation.

### Faculty Page
Displays teacher profiles and qualifications.

### Courses Page
Displays courses offered for different classes.

### Results Page
Displays previous year student results.

### Registration Page
Allows students to register for courses.

### Enquiry Form
Allows parents and students to ask questions.

---

## Integrations

Google Forms will be used to collect:

- Enquiries
- Registrations

Google Sheets will store the submitted data automatically.
