# Product Requirement Document (PRD)

## Product Overview

The product is an educational institute website designed to provide information about courses, faculty, results, and allow students to submit enquiries and registrations online.

---

## Target Users

Primary Users:

- Students
- Parents

Secondary Users:

- Institute administrators
- Faculty members

---

## Goals

The website aims to:

- Provide course information
- Display institute results
- Showcase faculty qualifications
- Allow students to register
- Collect enquiries from parents and students

---

## Features

### 1. Homepage
The homepage will introduce the institute and highlight key information such as courses, faculty, and results.

### 2. Course Programs
The website will provide information about different classes.

Programs include:

- Class 5th – 8th
- Class 9th – 10th
- Class 11th – 12th
- CET preparation

Supported boards:

- CBSE
- ICSE
- State Board

---

### 3. Faculty Section
Displays teacher profiles including:

- Name
- Qualification
- Experience
- Subject expertise

---

### 4. Results Section
Displays student results and achievements.

---

### 5. Registration Form
Allows students to register for courses.

Information collected:

- Name
- Phone
- Email
- Class
- Board

---

### 6. Enquiry Form
Parents and students can submit questions about courses.

Data will be stored using Google Forms integration.

---

### 7. Testimonials
Displays feedback from previous students and parents.

---

## Success Metrics

The success of the website will be measured by:

- Number of registrations
- Number of enquiries received
- Website traffic
